使用规范

标题规范

  • 页面标题使用 H1 (36px, bold)
  • 区块标题使用 H2 (30px, semibold)
  • 卡片标题使用 H3 (24px, semibold)
  • 表单标题使用 H4 (20px, medium)

文本规范

  • 正文使用 16px, normal
  • 辅助信息使用 14px, normal
  • 标签使用 12px, medium
  • 按钮文字使用 14px, medium
